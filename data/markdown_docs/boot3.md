

## 6. Bootstrap Badges

### What is a Badge?

A **Badge** is a small count or label used to highlight **status or numbers**.

### When to Use

* Notifications
* Status labels
* Counters

### Example Syntax

```html
<h5>
  Messages
  <span class="badge bg-primary">4</span>
</h5>

<span class="badge bg-success">Active</span>
<span class="badge bg-danger">Inactive</span>
```

## 7. Bootstrap Tables

### What is a Table?

A **Table** is used to display **structured data** in rows and columns.

### When to Use

* User lists
* Reports
* Data summaries

### Example Syntax

```html
<table class="table table-striped table-bordered">
  <thead>
    <tr>
      <th>#</th>
      <th>Name</th>
      <th>Role</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>1</td>
      <td>Eyobed</td>
      <td>Admin</td>
    </tr>
    <tr>
      <td>2</td>
      <td>Student</td>
      <td>User</td>
    </tr>
  </tbody>
</table>
```

## 8. Bootstrap Container

### What is a Container?

A **Container** is a responsive fixed-width wrapper that centers content.

### When to Use

* Main page layout
* Centered content
* Standard websites

### Example Syntax

```html
<div class="container">
  <h2>Fixed Container</h2>
  <p>This content is centered and responsive.</p>
</div>
```

## 9. Bootstrap Container-Fluid

### What is Container-Fluid?

A **Container-fluid** spans the **full width** of the viewport.

### When to Use

* Full-width layouts
* Dashboards
* Hero sections

### Example Syntax

```html
<div class="container-fluid bg-light">
  <h2>Full Width Container</h2>
  <p>This stretches across the screen.</p>
</div>
```

## 10. Bootstrap Grid System (Row & Columns)

### What are Rows and Columns?

Bootstrap uses a **12-column grid system** to create responsive layouts.

### When to Use

* Page layouts
* Responsive design
* Multi-column content

### Example Syntax

```html
<div class="container">
  <div class="row">

    <div class="col-md-6">
      <p>Left Column</p>
    </div>

    <div class="col-md-6">
      <p>Right Column</p>
    </div>

  </div>
</div>
```

### Responsive Example

```html
<div class="row">
  <div class="col-12 col-md-4">Column 1</div>
  <div class="col-12 col-md-4">Column 2</div>
  <div class="col-12 col-md-4">Column 3</div>
</div>
```
